🔄 Sync em tempo real

📖 Como configurar o Supabase (passo a passo)
1. Criar conta gratuita
Acesse supabase.com → clique em Start your project → faça login com GitHub.
2. Criar um projeto
Clique em New project → dê um nome (ex: "meu-tracker") → escolha a região mais próxima (South America) → defina uma senha e clique Create new project. Aguarde ~1 min.
3. Criar a tabela
No menu lateral, clique em SQL Editor → clique em New query → cole o código abaixo e clique Run: 
create table tracker_data (
  key text primary key,
  value jsonb,
  updated_at timestamptz default now()
);

alter table tracker_data enable row level security;

create policy "acesso total"
  on tracker_data for all
  using (true)
  with check (true);
4. Copiar suas credenciais
No menu lateral, clique em Project Settings → API. Você vai ver:
• Project URL → copie para o campo URL acima
• anon public (em Project API Keys) → copie para o campo Key acima
5. Conectar
Cole os dois valores nos campos acima e clique em Salvar e conectar. O tracker vai fazer o upload dos seus dados existentes automaticamente. Abra em outro dispositivo e os dados aparecem na hora! 🎉

📊 Resumo dos dados salvos

📤 Exportar

Gera um arquivo .json com todos os seus dados. Salve e transfira para outro dispositivo.

📥 Importar

⚠️ Importar vai substituir todos os dados atuais. Exporte primeiro se quiser guardar o que tem.

📋 Transferir via texto

Alternativa quando não dá para transferir arquivo. Copie o texto num dispositivo e cole no outro.
